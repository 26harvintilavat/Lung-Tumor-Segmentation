About the Project

LungSeg AI

A medical imaging pipeline and viewer designed for the automated segmentation of lung tumors from CT scans. It streamlines the diagnostic workflow by isolating nodules and computing volumetric metrics automatically.

Dataset

Powered by the LIDC-IDRI dataset from The Cancer Imaging Archive (TCIA). Consisting of diagnostic and lung cancer screening thoracic CT scans with marked-up annotated lesions.

Interpretability

All masks are generated using a 3D architecture tracking HU densities across the axial series. Detected volumes provide estimations critical for tracking morphological progression.

Model Architecture

  • Type: U-Net (PyTorch)
  • Input: Normalized CT Volume (HU mapped)
  • Output: Binary Segmentation Map (.npy)

Tech Stack

  • Frontend: Vanilla JS, HTML5 Canvas, CSS Variables
  • Viewer: Cornerstone.js Core
  • Parser: dicomParser
  • Backend: Python, FastAPI
